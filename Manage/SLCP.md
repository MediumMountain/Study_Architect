# ソフトウェアライフサイクルプロセス(SLCP)
- JIS X 160 とはソフトウェア・ライフサイクル・プロセスに関する日本工業規格で、システム開発の発注側と受注側の双方で相互に理解できるソフトウェア・ライフサイクル・プロセスの共通枠組みを、明確に定義された用語で規定したものである。プロセスおよびアクティビティ、タスクの3階層で構成される。
